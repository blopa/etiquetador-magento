# etiquetador-magento
Modulo de etiquetas para Magento baseado no modulo desenvolvido pelo Fabio Santos, que não estava funcionando depois do SUPEE-6788.

Fiz esse modulo para suprir a minha própria necessidade de imprimir etiquetas no Magento 1.9.2.2, esse modulo irá gerar 12 endereços por folha A4.
